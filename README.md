---

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte de um **desafio da DIO (Digital Innovation One)**.

O objetivo é criar um **assistente de voz inteligente** que permita ao usuário conversar com o ChatGPT utilizando fala.

### 🔄 Fluxo de funcionamento

O sistema segue o seguinte fluxo:

```
Usuário fala → Whisper transcreve → ChatGPT responde → gTTS gera voz
```

1. 🎤 O usuário fala no microfone
2. 🧠 **Whisper** converte a fala em texto
3. 🤖 **ChatGPT** processa a pergunta e gera uma resposta
4. 🔊 **gTTS** transforma a resposta em áudio

## 📖 About the Project

This project was developed as part of a **DIO (Digital Innovation One) challenge**.

The goal is to create an **AI voice assistant** that allows users to talk to ChatGPT using voice commands.

### 🔄 Workflow

The system follows this process:

```
User speaks → Whisper transcribes → ChatGPT responds → gTTS generates voice
```

1. 🎤 The user speaks into the microphone
2. 🧠 **Whisper** converts speech into text
3. 🤖 **ChatGPT** processes the request and generates a response
4. 🔊 **gTTS** converts the response into audio
