# Voice Assistant com ChatGPT + Whisper

Projeto desenvolvido no desafio da DIO.

Permite conversar com o ChatGPT usando voz.

Fluxo:

Usuário fala → Whisper transcreve → ChatGPT responde → gTTS gera voz.

Tecnologias:

Python
Whisper
OpenAI API
gTTS
